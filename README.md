# AI Product Store

Website bán sản phẩm AI - Landing Page + Admin Dashboard kế toán

## Cài đặt

```bash
npm install
npm run seed   # Tạo database và dữ liệu mẫu
npm run dev    # Chạy development mode
```

## Truy cập
- Landing Page: http://localhost:3000
- Admin: http://localhost:3000/admin
- Tài khoản mặc định: admin / admin123
